
# Dmoz crawler

## Instalation

You need clone application :

```sh
$ cd workspace #directory where will be current project
$ git clone https://git@github.com:valexl/crawler_dmoz.git
$ cd crawler_sketch
```

You need make bundle install before:

```sh
$ bundle install
```

Test that everything is well:

```sh
$ rspec spec
```

Try it in irb:

```sh
$ irb -r ./boot.rb
```
